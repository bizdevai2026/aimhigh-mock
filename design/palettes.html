How to read these

  1. Mini-hero in each card uses the same copy your kid sees on day 1, so the comparison is honest, not abstract.
  2. Swatch row shows the five colour roles each palette uses (background, text, primary, secondary, strong/headline).
  3. Type sample shows the display headline + one body paragraph in the option's chosen pairing.
  4. Flame treatment is animated; reload if you want to see the entrance again. Each treatment is independent of the palette — they can be mixed if you find a colour you like in one option but a flame style from another.

To pick: tell me the option number you want (1–5) and whether you want to keep its flame treatment or swap one in. I'll lift those tokens into mock.css and fonts into index.html across all 7 pages.
