# brand/

Four interactive previews of the GradeBlaze rebrand directions. Each
file is a self-contained HTML page that re-skins a representative
sample of the app (header + hero card + streak + tiles + buttons +
typography + palette) so the directions can be compared side by side
on real-rendered screens rather than described in words.

These pages are exploration only. Nothing here is wired into the live
app. Once a direction is chosen, the brand/ folder will be removed
and the chosen palette + typography applied to the actual app via
mock.css and the HTML titles/wordmarks.

## How to view

**Locally (laptop)**: open `brand/index.html` in any browser by
double-clicking it from File Explorer.

**Live (any device)**: GitHub Pages serves these too —
<https://bizdevai2026.github.io/aimhigh-mock/brand/>

Each option is one click from the index page.

## To pick a direction

Reply in chat with `brand option 1` (or 2 / 3 / 4). I'll then do the
full spec — CSS variable changes, logo SVG, sample screen render —
before any actual app file is touched.
